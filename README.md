🤖 AI Model Hub

AI Model Hub is a modern subscription-based web application UI where users can explore different AI models and their features. The platform is designed with a clean interface and smooth user experience for showcasing AI tools.

  Live Demo

🔗 https://ai-model-hub-subscription-website.netlify.app/

 GitHub Repository

🔗 https://github.com/probillal/ai-model-hub

✨ Features

* AI model listing and exploration UI
* Subscription-based layout design
* Fast and responsive interface
* Modern and clean UI design
* Fully mobile responsive
  
🛠️ Tech Stack
React.js
Tailwind CSS
JavaScript (ES6+)
React Router DOM

⚙️ Installation & Setup
# Clone the repository
git clone https://github.com/probillal/ai-model-hub.git

# Navigate to project folder
cd ai-model-hub

# Install dependencies
npm install

# Run development server
npm run dev
